# Plåtfinish Skellefteå AB – statisk onepager (HTML/CSS/JS)

## Innehåll
- Mörkt tema, modern känsla
- Scroll-snap (”swipe/scrolla ned -> ny sektion”)
- Smooth transitions (reveal animations)
- Mobilmeny + desktopnav + dot-nav
- Kontaktform utan backend (öppnar mailto)

## Starta lokalt
Öppna `index.html` direkt i webbläsare, eller kör en enkel server:

### Python
```bash
python -m http.server 8080
```
Öppna sedan http://localhost:8080

## Byt bilder
- Lägg era projektbilder i `assets/` och uppdatera galleriet i `index.html`.
